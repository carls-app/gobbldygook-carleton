The Department Class Specification.

This class models a department at St Olaf College.

Xandra Best. Wednesday, May 15, 2013




CLASS Department

No superclass.

State variables:

private dept_t id
Department ID.

Constructors:

Department
Default constructor.

No arguments.

State change:
Department object is initialized with id = NONE.

Department object is initialized with corresponding id.

Represents a department.

Department
