The Major Class Specification

This class models a major at St Olaf College.

Xandra Best. Wednesday, May 15, 2013




CLASS Major

No superclass.

State variables:

string name
Name of major.

Department department
Associated department of major.

vector of MajorRequirement requirements
Regular requirements in major.

vector of SpecialRequirement specialRequirements
Special requirements in major.

vector of MajorRequiremnet setRequirements
Regular requirements in Special requirements.

Constructors:

Major
Default constructor.

No arguments.

State change:
This instance of Major is initialized by setting state varible name to "None" and department to "NONE".

Major
Regular constructor.

Arguments:

string str
Name of major.

State change:
This instance of Major is initialized by calling init.

Major
Copy constructor.

Arguments:

const Major c
Reference to a major.

State change:
This instance of Major is initialized as a copy of c by calling copy.

Methods:

copy
Creates a copy of another Major

Arguments:

const Major c
Reference to a major.

State change:
No state change.

Return value:
An exact copy of Major c.

init
Initializes instance of Major.

Arguments:

vector of string record
A group of majors.

State change:
Instances of Major are initialized.

=
Equals operator.

Arguments:

const Major c
Reference to a major.

State change:
No state change.

Return value:
Reference to c.

ostream getData
Read in data.

Arguments:

ostream os
Reference to ostream.

State change:
Data in os is read into state variable name.

Return value:
os is returned.

display
Couts name.

No arguments.

State change:

Return value:
Couts name of major.

Searches through requirement vectors and returns relevant pointer.

Arguments:

Name of special requirement.

State change:
No state change.

Return value:
Pointer to the SpecialRequirement object.

MajorRequirement getMajorRequirement
Searches through requirement vectors and returns relevant pointer.

Arguments:

string str
Name of major requirement.

State change:
No state change.

Return value:
Pointer to the MajorRequirement object.

SetRequirement getSetRequirement
Searches through requirement vectors and returns relevant pointer.

Arguments:

string str
Name of set requirement.

State change:
No state change.

Return value:
Pointer to the SetRequirement object.
