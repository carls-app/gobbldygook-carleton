The SpecialRequirement Class Specification

This class models a special major requirement.

Xandra Best. Wednesday, May 15, 2013




CLASS SpecialRequirement

Superclass Requirement

State variables:

vector of MajorRequirement validSets
Vector of sets of requirements that fulfill this requirement.

Constructors:

SpecialRequirement
Default constructor.

No arguments.

State change:
SpecialRequirement object is initialized by calling Requirement default constructor.

SpecialRequirement
Partial constructor.

Arguments:

string str
Represents name of requirement.

State change:
SpecialRequirement object is initialized by calling Requirement partial constructor.

SpecialRequirement
Regular constructor.

Arguments:

string str
Represents name of requirement.

int n
Represents number of courses needed to fulfill this requirement.

State change:
SpecialRequirement object is initialized by calling Requirement regular constructor.

SpecialRequirement
Copy constructor

Arguments:

const SpecialRequirement c
Reference to a SpecialRequirement.

State change:
SpecialRequirement object is initialized by calling Requirement copy constructor.
