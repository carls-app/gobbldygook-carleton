The Student Class Specification

This class models a student at St Olaf College

Xandra Best. Wednesday, May 15, 2013




CLASS Student

No superclass.

State variables:

vector of Major majors
The student's majors.

vector of Concentration concentrations
The student's concentrations.

vector of Course courses
The courses taken by a student.

Constructors:

Student
Default constructor.

No arguments.

State change:
This instance of Student is initialized by calling init, where n="", s=2000, g=2004, m="".

Student
Manual constructor.

Arguments:

string n
The name of the student.

int s
The starting year of the student.

int e
The graduating year of the student.

string m
The majors of the student

State change:
This instance of Student is initialized by calling init, where n=n, s=s, g=e, m=m.

Student
Read-in constructor.

Arguments:

string fn
File name.

State change:
This instance of Student using data from file.

Methods:

private init
Initializes Student object.

Arguments:

string n
The name of the student.

int s
The starting year of the student.

int g
The graduation year of the student.

string m
The majors of the student.

State change:
Student object is initialized with name=n, startingYear=s, graduationYear=g, and majors is filled in with m.

Return value:
No return value.

bool hasTakenCourse
Checks to see if student has taken a given course

Arguments:

string str
The course to be checked.

State change:
No state change.

Return value:
Bool representing whether the student has taken the course.

addCourse
Add a course to courses student has taken.

Arguments:

const Course c
A reference to a course.

State change:
The given course is added to state variable courses.

Return value:
No return value.

addCourses
Adds multiple courses to courses student has taken.

Arguments:

string str
Comma separated list of courses.

State change:
The given courses are added to state variable courses.

Return value:
No return value.

addMajor
Add a major to student's majors

Arguments:

const Major m
A reference to a major.

State change:
The given major is added to state variable majors.

Return value:
No return value.

addMajors
Adds multiple majors to student's majors.

Arguments:

string str
Comma separated list of majors.

State change:
The given majors are added to state variable majors.

Return value:
No return value.

addConcentration
Add a concentration to student's concentrations

Arguments:

const Concentration m
A reference to a concentration.

State change:
The given concentration is added to state variable concentrations.

Return value:
No return value.

addConcentrations
Adds multiple concentration to student's concentrations.

Arguments:

string str
Comma separated list of concentrations.

State change:
The given concentrations are added to state variable concentrations.

Return value:
No return value.

ostream getData
Relays data about students.

Arguments:

ostream os
reference to ostream.

No state change.

Reference to ostream detailing student's majors, concentrations, and courses.

displays student's majors, concentrations, and courses.
