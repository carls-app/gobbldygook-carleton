The Course Class Specification

This class models a course at St Olaf College.

Xandra Best. Wednesday, May 15, 2013




CLASS Course

No superclass.

State variables:

ID id
Course id.

string title
Title of course.

string description
Description of course.

vector of Major majors
Majors associated with course.

vector of Department department
Departments associated with course.

string concentrations
Concentrations associated with course

string conversations
Conversations associated with course.

string professor
Professor of course.

int half_semester
Whether course is half-semester.

bool pass_fail
Whether course is pass-fail.

float credits
Number of credits with course.

string location
Location of classes.

course_type_t courseType
Type of course. (L/T/S)

GenEd geneds
GenEds associated with course.

bool days
Array of days of classes.

float time
Array of times of classes.

Constructors:

Course
Default constructor.

No arguments.

State change:
Initializes null Course object with reasonable default values.

Course
Regular constructor.

Arguments:

string str
ID of course.

State change:
Initializes Course object by calling init.

Course
Copy constructor.

Arguments:

const Course c
Reference to a course.

State change:
Initializes Course object as a copy of the given course.

Course
Read-in constructor.

Arguments:

istream is
Reference to istream.

State change:
Initializes Course object with data read in from istream.

Methods:

private init
Course initializer.

Arguments:

string identifier
Course ID.

State change:
Initializes instance of Course object.

Return value:
No return value.

private copy
Makes Course object a copy of another course.

Arguments:

const Course c
Reference to a Course object.

State change:
sets all state variables equal to state variables of c.

Return value:
No return value.

Course =
Assignment operator.

Arguments:

const Course c
Reference to Course object.

State change:
State variables are changed to match those of Course c.

Return value:
Reference to copy of c.

friend bool ==
Equal to operator.

Arguments:

const Course c1
Reference to first Course object.

const Course c2
Reference to second Course object.

State change:
No state change.

Return value:
True if courses are identical.

friend bool !=
Not equal to operator

Arguments:

Course c1
Reference to first Course object.

Course c2
Reference to second Course object.

State change:

Return value:
True if courses are not identical.

friend Course getCourse
Identifies and returns course.

Arguments:

string identifier
Course ID.

State change:
No state change.

Return value:
Course object associated with identifier, or null course if not found.

cleanTitle
Cleans course title.

No arguments.

State change:
Sets title to a cleaner version.

Return value:
No return value.

Department getDepartment
Get department.

Arguments:

int i
Index of department vector.

State change:
No state change.

Return value:
Returns specified Department object

string getProfessor
Get professor.

No arguments.

State change:
No state change.

Return value:
Returns professor.

ID getID
Get ID.

No arguments.

State change:
No state change.

Return value:
Returns ID.

string getType
Get course type.

No arguments.

State change:
No state change.

Return value:
Returns string of course type.

int getNumber
Get ID number.

No arguments.

State change:
No state change.

Return value:
Returns number part of ID.

string getSection
Get ID section.

No arguments.

State change:
No state change.

Return value:
Returns section part of ID.

ostream getData
Get course type, ID, title, professor.

Arguments:

ostream os
Reference to ostream.

State change:
No state change.

Return value:
Returns reference to ostream containing this data.

display
Displays data on Course object.

No arguments.

State change:
No state change.

Return value:
Couts course type, ID, title, professor.

showAll
Displays all data on Course object.

Arguments:

ostream os
Reference to ostream.

State change:
No state change.

Return value:
Couts id, title, prof, type, half_semester, credits, pass_fail, location.
